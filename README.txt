موسوعة RHEL العربية — واجهة Knowledge UI V1

الملفات المطلوبة للنشر:
- index.html
- style.css
- app.js
- knowledge.json

طريقة التشغيل محلياً:
1. افتح موجه الأوامر داخل المجلد.
2. نفّذ:
   python -m http.server 8000
3. افتح:
   http://localhost:8000

طريقة التحديث على GitHub:
- استبدل ملفات index.html وstyle.css وapp.js القديمة.
- أضف knowledge.json إلى جذر المستودع.
- يمكن إبقاء commands.json مؤقتاً، لكنه لم يعد مستخدماً في الواجهة الجديدة.

الواجهة تدعم:
- عرض Workflows متعددة الخطوات.
- استبدال المتغيرات داخل الأوامر.
- التحقق، الأخطاء الشائعة، التراجع، الملفات والمنافذ.
- حفظ المفضلة وتقدم الخطوات والوضع الليلي في المتصفح.
- روابط مباشرة للمهام بواسطة #task=TASK_ID.
